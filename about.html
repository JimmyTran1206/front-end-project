1.Project Info

Aesthetik is a front-end web project developed as a mid-term project at LearningFuze Coding Bootcamp to showcase my skills in building responsive, interactive websites using only plain HTML, CSS, and JavaScript, without relying on any external frameworks or libraries. Designed with a modern aesthetic and intuitive user experience in mind, Aesthetik simulates a shopping platform where users can browse products, view details, and manage a cart.

The project emphasizes clean layout structuring with semantic HTML, visually appealing and responsive design using CSS, and dynamic functionality powered by vanilla JavaScript. Through this project, I aimed to demonstrate a solid understanding of core web development principles, DOM manipulation, and user interaction handling. skills in plain HTML, CSS, and JavaScript.

Key features include:

  • Product listing and detail views
  • Shopping cart functionality
  • Responsive layout for various screen sizes
  • Smooth UI interactions and transition

Aesthetik represents my ability to create fully functional and visually polished front-end applications without the use of frameworks—relying purely on foundational web technologies.



2.Technology Used

This project showcases a modern, minimalistic e-commerce interface where users can browse products, view item details, and manage a shopping cart—all developed from scratch using foundational web technologies:

  • HTML5: structure all page tags and elements
  • CSS3: styling all elements using one single stylesheet with emphasize on responsiveness across various screen size
  • TypeScript-JavaScript-ES6: TypeScript for type checking, which is transpiled into JavaScript at runtime. The code is organized into reusable modules that serve specific purposes, like carts, filter, and pages
